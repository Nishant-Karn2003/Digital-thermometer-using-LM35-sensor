# 🌡️ Digital Thermometer using LM35 Sensor and Arduino

---

# 📌 Overview

This project implements a simple and efficient **Digital Thermometer** using an **LM35 temperature sensor** and an **Arduino microcontroller**.

The LM35 provides an analog voltage proportional to temperature, which is processed by the Arduino and displayed on a **16×2 LCD display** or through the serial monitor.

The system is **low-cost, accurate, and power-efficient**, making it suitable for applications such as:

* Home automation
* Weather monitoring
* Healthcare systems
* Industrial temperature control

# ⚡ Working Principle

1. The **LM35 sensor** outputs an analog voltage proportional to temperature.

2. The sensor produces:

   * **10mV output per °C**

3. Arduino reads the analog voltage from the LM35 sensor pin.

4. The microcontroller converts the voltage into temperature values in **degree Celsius (°C)**.

5. The measured temperature is displayed on:

   * 16×2 LCD Display
   * Serial Monitor

6. An optional **buzzer alert system** can be activated if the temperature exceeds a predefined threshold.
